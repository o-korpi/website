About this website

This website was built using Go, Templ, HTMX and surreal.js. HTMX and Surreal bring interactivity to the website, while Templ handles the templating.

The blog pages are created by rendering Markdown and converting it to HTML. Inline LaTeX support is added using some custom parsing and by using MathJax on the frontend.

These features are for the full work-in-progress blog system, allowing for automatic rendering of markdown files.
